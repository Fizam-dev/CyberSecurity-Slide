Network Security

INFRASTRUCTURE & PERIMETER

Kontrol & Hardening

Mekanisme pertahanan utama untuk melindungi integritas jaringan.

Segmentasi Jaringan

Membagi jaringan menjadi zona-zona terisolasi (VLAN) untuk membatasi penyebaran serangan (lateral movement).

Firewall & IDS/IPS

Next-Gen Firewall untuk filtering trafik dan Intrusion Detection/Prevention Systems untuk mendeteksi pola serangan.

Zero Trust & NAC

Network Access Control (NAC) dan prinsip "Never trust, always verify" untuk setiap koneksi.

Ancaman & Pengujian

Jenis serangan umum dan cara memvalidasi keamanan.

Common Threats

  • DDoS Attacks

  • MITM (Man-in-the-Middle)

  • IP Spoofing

  • Packet Sniffing

Metode Pengujian (Testing)

Vulnerability Scanning

Pemindaian otomatis port dan layanan (Nmap, Nessus).

Penetration Testing

Simulasi serangan etis untuk mengeksploitasi celah.
